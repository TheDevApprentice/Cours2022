Accès aux éléments d'une page et modification

Rubriques correspondantes sur le site du w3schools

  • accès aux éléments:
    • à l'aide du id (getElementById)
    • à l'aide d'une balise (getElementsByTagName)
    • à l'aide d'un nom de classe (getElementsByClassName)
    • à l'aide d'un sélecteur css (querySelector et querySelectorAll)
  • modifier du code HTML et des attributs(.innerHtml et .nomAttribut)
  • modifier le CSS (.style.nomPropriété
